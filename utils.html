utils.get_unexpanded_path(path)[source]¶

if start by /home/foo, convert /home/foo by ~ in a path

>>> get_unexpanded_path('/home/gnu/dir') 
'~/dir'
>>> get_unexpanded_path('/tmp/foo')
'/tmp/foo'
>>> get_unexpanded_path('/home/fr/foo/home/bar')
'~/foo/home/bar'
